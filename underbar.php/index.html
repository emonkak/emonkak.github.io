Underbar.phpは配列操作のための便利な関数群を提供するUnderscore.jsライクなPHPのライブラリです。

API

Collection Functions

void each(array $xs, callable $f)

arrayの各要素に対して関数$fを呼び出します。 $fは以下の3つの引数を取ります。

$x
配列の要素
$i
配列の添字
$xs
配列自身
>>> _::each([1, 2, 3], function($x, $i, $xs) {
  echo $x, PHP_EOL;
})
=> 1
=> 2
=> 3
>>> _::each(['one' => 1, 'two' => 2, 'three' => 3], function($x, $k, $xs) {
  echo $k, $x, PHP_EOL;
})
=> one1
=> two2
=> three3
TOP

array map(array $xs, callable $f)

Arrayの各要素に対して関数$fを適用して新しい配列を返します。 $fは以下の3つの引数を取ります。

$x
配列の要素
$i
配列の添字
$xs
配列自身
>>> _::map([1, 2, 3], function($x) {
  return $x * 3;
})
=> [3, 6, 9]
>>> _::map(['one' => 1, 'two' => 2, 'three' => 3], function($x) {
  return $x * 3;
})
=> [3, 6, 9]
TOP

Array Functions

array first(array $xs)

mixed first(array $xs, int $n)

  • head
  • take

配列の最初の要素を返します。 $nを指定した場合は先頭からn個分の要素を含む配列を返します。 

>>> _::first([1, 2, 3])
1
>>> _::first([1, 2, 3], 1)
[1]
>>> _::first([1, 2, 3], 2)
[1, 2]
