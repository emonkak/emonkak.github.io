`$xs`の要素の中の_最大_の値を返します。
`$f`に関数を指定した場合は、その関数を適用した結果を基準にします。
`$f`に文字列を指定した場合は、該当のプロパティを基準にします。
`$xs`が空の時は`-INF`を返します。

```php
$stooges = [
    ['name' => 'moe', 'age' => 40],
    ['name' => 'larry', 'age' => 50],
    ['name' => 'curly', 'age' => 60]
];
_::max($stooges, 'age');
=> ["name" => "curly", "age" => 60]
```
